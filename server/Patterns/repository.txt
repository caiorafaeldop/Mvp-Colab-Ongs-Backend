PADRÃO REPOSITORY (Estrutural)

CONTEXTO NO PROJETO:
- Abstração da camada de dados
- Centralização de queries do MongoDB
- Facilita testes com mock repositories

APLICAÇÃO PRÁTICA:
1. UserRepository - operações CRUD de usuários/ONGs
2. CollaborationRepository - gerencia parcerias entre ONGs
3. ProjectRepository - repositório de projetos das ONGs
4. FileRepository - gerencia uploads e arquivos
5. NotificationRepository - histórico de notificações

BENEFÍCIOS:
- Desacoplamento entre domínio e persistência
- Facilita testes unitários
- Centraliza lógica de acesso a dados
