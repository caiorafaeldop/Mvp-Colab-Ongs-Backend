PADRÃO DECORATOR (Estrutural)

CONTEXTO NO PROJETO:
- Adicionar funcionalidades extras aos controllers (logging, cache, validação)
- Middleware de autenticação e autorização
- Enriquecimento de dados de resposta

APLICAÇÃO PRÁTICA:
1. LoggingDecorator - adiciona logs automáticos aos métodos
2. CacheDecorator - adiciona cache às consultas
3. ValidationDecorator - adiciona validação automática
4. AuthDecorator - adiciona verificação de autenticação
5. RateLimitDecorator - adiciona controle de taxa de requisições

BENEFÍCIOS:
- Adiciona responsabilidades sem modificar código original
- Composição flexível de funcionalidades
- Reutilização de decoradores
