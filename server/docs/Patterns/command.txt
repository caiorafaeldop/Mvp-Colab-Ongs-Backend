PADRÃO COMMAND (Comportamental)

CONTEXTO NO PROJETO:
- Operações que podem ser desfeitas (edição de perfil, colaborações)
- Fila de tarefas assíncronas (processamento de uploads, emails)
- Histórico de ações do usuário

APLICAÇÃO PRÁTICA:
1. UploadCommand - encapsula operação de upload com rollback
2. CollaborationCommand - comandos para iniciar/cancelar parcerias
3. NotificationCommand - comandos de envio de notificações
4. ReportCommand - geração de relatórios como comando
5. BackupCommand - comandos de backup e restore

BENEFÍCIOS:
- Operações podem ser desfeitas
- Facilita implementação de filas
- Histórico de comandos executados
